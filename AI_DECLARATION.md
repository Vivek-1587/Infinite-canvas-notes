# AI Declaration

## AI Tools Used

* ChatGPT

## How AI Was Used

This project was developed with significant assistance from ChatGPT.

AI was used for:

* Project planning and feature breakdown
* Architecture design
* Next.js and TypeScript code generation
* React component generation
* IndexedDB integration guidance
* Graph view implementation guidance
* Export functionality implementation
* Documentation generation
* Testing setup and debugging assistance

## Development Process

The project was developed through iterative conversations with ChatGPT. Requirements were provided, code was generated and refined over multiple interactions, and the resulting codebase was assembled, tested, committed to GitHub, and deployed by the developer.

## Human Contributions

The developer:

* Defined the project requirements
* Reviewed generated code
* Managed project files
* Performed Git operations
* Uploaded the project to GitHub
* Deployed the application
* Tested functionality and resolved deployment issues

## Responsibility

The developer reviewed and submitted the final project and takes responsibility for the final repository contents.
